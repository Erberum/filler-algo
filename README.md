# filler-algo

Algorithmic solver for the filler-game (>3e+32 potential game states)

# Installation

1. Install dependencies through poetry

```bash
poetry install
```

2. Activate the environment

```bash
source $(poetry env info -p)/bin/activate
```

3. Build

```bash
python setup.py build_ext -i
```

# Running

```bash
python main.py
```
